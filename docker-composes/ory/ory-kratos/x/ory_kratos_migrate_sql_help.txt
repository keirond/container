Usage:
  kratos migrate sql <database-url> [flags]
  kratos migrate sql [command]

Available Commands:
  down        Rollback the last applied SQL migrations
  status      Display the current migration status
  up          Apply all pending SQL migrations

Flags:
  -c, --config strings   Path to one or more .json, .yaml, .yml, .toml config files. Values are loaded in the order provided, meaning that the last config file overwrites values from the previous config file.
  -h, --help             help for sql
  -e, --read-from-env    If set, reads the database connection string from the environment variable DSN or config file key dsn.
  -y, --yes              If set all confirmation requests are accepted without user interaction.

Use "kratos migrate sql [command] --help" for more information about a command.
